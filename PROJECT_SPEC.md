## 1. プロジェクト概要

### サイト名
**つづくウェルネス（TSUZUKU WELLNESS）**

### タグライン
「からだが変わる、習慣がつづく。」

### サブコピー
「続けるから、変わる。パーソナルジム RESIST が届けるウェルネスメディア」

### サイトの目的
- RESISTのブランド認知を「ジム検討中」以前の潜在層にまで拡張する
- SEO・SNS経由でウェルネス関心層を集客し、RESISTへの導線を構築する
- 科学的根拠に基づくウェルネスコンテンツで信頼性を確立する（E-E-A-T強化）

### ターゲット
| 層 | 属性 | ニーズ |
|---|---|---|
| 潜在層 | 運動習慣なし・不調を感じている | セルフケア情報、習慣づくりのヒント |
| 準顕在層 | ジム・ピラティスに興味はあるがまだ検討前 | トレーニング方法、ビフォーアフター |
| 顕在層 | ジム比較中・体験を検討中 | 口コミ、体験ストーリー、料金情報 |

## 2. 参考サイト分析

### Good Life Meds (https://www.goodlifemeds.com/)

**サイト構造の特徴:**
- ヒーローセクション: 大きなビジュアル＋メインコピー＋CTA
- カテゴリナビゲーション: Weight Loss / Daily Wellness / Sexual Health / Hair
- 信頼性バッジ: "Trusted by 100K+", "FDA-Regulated", "Board certified"
- How it Works: 3ステップの簡潔なフロー説明
- 商品カード: 横スクロール型で各カテゴリの商品を紹介
- Health Guide: ブログ記事セクション
- FAQ: アコーディオン形式
- フッター: カテゴリ、リソース、連絡先、ニュースレター登録

**つづくウェルネスに応用するポイント:**
| Good Life Medsの要素 | つづくウェルネスでの応用 |
|---|---|
| 商品カード（横スクロール） | 記事カード（カテゴリ別） |
| "Trusted by 100K+" | 「RESIST会員の〇〇%が実感」等の実績表示 |
| How it Works 3ステップ | 「RESISTを始めるまでの3ステップ」 |
| Health Guide（ブログ） | メインコンテンツ（記事一覧） |
| カテゴリナビ（Mega Menu） | 記事カテゴリナビ |
| 商品CTA「Get Started」 | 記事CTA「体験予約」「LINE登録」 |

**デザイン面で参考にする要素:**
- クリーンな白基調のレイアウト
- セクションごとの明確な区切り（背景色の切り替え）
- カード型UIのプロダクト表示
- モバイルファーストのレスポンシブ設計
- スムーズなアニメーション・トランジション

## 6. CV（コンバージョン）設計

### ファネル構造

```
【認知・流入】          【関係構築】           【CV・来店】
  記事閲覧       →     LINE登録/DL     →     体験予約/来店
  (SEO/SNS)          (リードナーチャリング)       (成約)
```

### CVポイント一覧

| CVポイント | 配置場所 | ターゲット層 |
|---|---|---|
| LINE公式登録 | 記事内CTA、ポップアップ | 潜在層〜準顕在層 |
| 無料PDFダウンロード | 特集ページ、記事内 | 準顕在層 |
| ウェルネス診断 | トップページ、専用ページ | 潜在層 |
| 体験予約 | フローティングCTA、記事末尾 | 顕在層 |

### CTA文言ガイドライン

```
✅ 推奨表現:
  - 「まずは1回、体験してみる →」
  - 「この記事の内容を、トレーナーと一緒に実践する →」
  - 「あなたの"つづく"を、一緒に見つけませんか？」
  - 「つづくウェルネスのLINEで毎週1つ、カラダが変わる習慣をお届け」

❌ 避ける表現:
  - 「今すぐ入会！」（ハードルが高い）
  - 「限定○名！急いで！」（煽り感）
```

### KPI設計

| 指標 | 目標値（月次） | 備考 |
|---|---|---|
| PV | 初期3ヶ月: 1万PV → 6ヶ月: 5万PV | SEO記事蓄積で段階成長 |
| LINE登録率 | PVの2〜3% | 記事CVRの目安 |
| 体験予約率 | LINE登録者の5〜10% | ステップ配信経由 |
| 体験→入会率 | 50%以上 | RESIST既存の強みを活用 |

## 7. SEO戦略

### コンテンツカテゴリ × キーワード方針

| カテゴリ | ターゲットKW例 | 記事タイプ |
|---|---|---|
| トレーニング | 「筋トレ 初心者」「自宅トレーニング」 | ハウツー、メニュー紹介 |
| ピラティス | 「ピラティス 効果」「マシンピラティス」 | 解説、体験レポート |
| 食事・栄養 | 「ダイエット 食事」「タンパク質 摂り方」 | 栄養学解説、レシピ |
| セルフケア | 「ストレッチ 腰痛」「肩こり 解消」 | 実践ガイド |
| 習慣づくり | 「運動 続ける コツ」「朝活 メニュー」 | マインドセット、Tips |
| 体験ストーリー | 「パーソナルジム 体験談」 | インタビュー、ビフォーアフター |

### ローカルKW方針
「地名がある理由になる記事」にのみ使用する原則を適用。
- ✅ 「北浦和 パーソナルジム 体験」（RESIST店舗紹介記事）
- ✅ 「西新宿 ピラティス 初心者」（RESIST店舗紹介記事）
- ❌ 一般的なハウツー記事にローカルKWを無理に入れない

### 構造化データ（JSON-LD）

各ページタイプに応じた構造化データを実装:
- トップページ: `Organization` + `WebSite`
- 記事ページ: `Article` + `BreadcrumbList` + `Person`（監修者）
- FAQセクション: `FAQPage`

## 11. ロゴ・ブランディング

### カラー方針
RESIST公式サイトのカラーイメージを踏襲:
- 白基調・クリーンなデザイン
- アクセントカラー: オレンジ(#FF6B35)→赤(#E63946)のグラデーション

### ロゴ方向性（候補）
1. **ループ・循環モチーフ型**: 「つ」を∞ループ化
2. **手書き風ナチュラル型**: 温かみ・親しみやすさ重視
3. **英字モダンブランド型**: 「TSUZUKU」をメインに
4. **植物×幾何学型**: 葉のモチーフでウェルネス感
5. **ライフライン波形型**: 心拍・呼吸のリズムを表現
6. **積み重ね・成長モチーフ型**: ドットの段階的成長
