# つづくウェルネス（TSUZUKU WELLNESS）

> からだが変わる、習慣がつづく。

パーソナルジム＆ピラティススタジオ **RESIST** が運営するウェルネスメディアサイト。

## Tech Stack

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS** (utility-first styling)
- **Framer Motion** (animations)
- **React Router** (SPA routing)
- **Vercel** (hosting)

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
tsuzuku-wellness/
├── public/                  # Static assets
├── src/
│   ├── assets/             # Images, icons
│   ├── components/         # Reusable UI components
│   ├── data/               # Site data, articles, config
│   ├── pages/              # Route-level page components
│   ├── styles/             # Global CSS
│   ├── types/              # TypeScript type definitions
│   ├── App.tsx             # App root with routing
│   └── main.tsx            # Entry point
├── PROJECT_SPEC.md          # Site design specification
├── package.json
├── tailwind.config.js
├── tsconfig.json
└── vite.config.ts
```
